### fastapi-sample

**Stack:** FastAPI 0.115 · Python 3.12 · pytest · pip + venv

**What it demonstrates:**
- FastAPI app with root, list, create, get, and delete item routes
- `requirements.txt` / `requirements-dev.txt` split for prod vs dev dependencies
- `requirements.lock` for reproducible installs (generated by `pip freeze`)
- Sync `TestClient` tests with pytest covering happy paths and 404 errors
- `.python-version` for pyenv version pinning

**How to run:**
```bash
cd fastapi-sample

# Install Python 3.12 via pyenv (if not already installed)
pyenv install 3.12
pyenv local 3.12

# Create virtual environment and install dependencies
python3 -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate

pip install -r requirements-dev.txt

# Run tests
pytest

# Start the development server
uvicorn app.main:app --reload
```
